¿Qué es un condicional?

Los condicionales con instrucciones que controlan el flujo de ejecución de tu programa, depende de si la condición es True (verdadero) o False (falso). Permite que el código se adapte a diferentes situaciones.

Las instrucciones básicas en python son: if (si) que permite especificar una condición, y si esa condición se evalúa como True (verdadero), se ejecutará un bloque de código. La instrucción else (sino), proporciona una alternativa si la ejecución en la instrucción if es False (falso). Y por último la instruccion elif (si no) que permite verificar múltiples condiciones secuencialmente. Se usa dentro de una estructura if y te permite definir condiciones adicionales para ejecutar diferentes bloques de código.

Ejemplos


    calificacion = 70
    
    if calificacion >= 90:       #Este bloque se imprime si es mayor o igual a 90.
        print("Excelente!")    

    elif calificacion >= 80:     #Este bloque se imprime si es mayor o igual a 80.
        print("Muy bien!")

    else:                        #Si ninguno de los alteriodes es True imprime este bloque.
        print("Sigue practicando!")

En este ejemplo, la instrucción if primero verifica si el puntaje es mayor o igual a 90. Si no se cumple, la instrucción elif verifica si el puntaje es mayor o igual a 80. Finalmente, si ninguna condición es verdadera, se ejecuta else.

Los condicionales se basan en operadores de comparación para evaluar condiciones. Estos operadores comparan valores y devuelven True o False. Algunos operadores de comparación comunes incluyen:

  • == (igual a)
  • != (diferente a)
  • < (menor que)
  • > (mayor que)
  • <= (menor o igual que)
  • >= (mayor o igual que)

    • Igualmente estan los operadores lógicos que permiten combinar expresiones booleanas (True o False) para crear condiciones más complejas. Estos operadores son esenciales para controlar el flujo de tu programa y tomar decisiones basadas en diferentes escenarios.

    Tipos de operadores lógicos:

  • and (y): Este operador devuelve True solo si ambas expresiones son True
    • 
    edad_mayor_igual_18 = True
    tiene_identificacion = True
    
    puede_entrar = edad_mayor_igual_18 and tiene_identificacion
    
    print(entrar)               #En este caso como cumple las 2 condiciones puede entrar
  • or (o): Este operador devuelve True si al menos una expresión es True
    • 
    tiene_dni = True
    tiene_passaporte = False
    
    puede_entrar = edad_mayor_igual_18 and tiene_identificacion
    
    print(entrar)               #En este caso como cumple las 2 condiciones puede entrar
  • not (no): Este operador invierte el valor de True de una expresión. 
    • 
    registrado = False
    
    no_registrado = not registrado
    
    print(no_registrado)            #En este caso imprime True porque esta invirtiendo el valor False que tenia la variable.

    ¿Cuáles son los diferentes tipos de bucles en Python? ¿Por qué son útiles?

    Entre los diferentes tipos de bucles en python se encuentra for, que se utiliza para iterar sobre una secuencia de valores, como una lista o una cadena. Es útil para automatizar tareas repetitivas que se ejecutan un número determinado de veces.

    
    numeros = [1, 2, 3, 4, 5]

    for numero in numeros:
        print(numero)        #Este bucle va a buscar en la lista de numeros, el numero que queremos imprimir

    Tambien esta el bucle while que se utiliza para ejecutar un bloque de código mientras una condición sea True, sino el bucle no sabe cuando finalizar y va a repetir la accion hasta dar error. Es útil para automatizar tareas repetitivas que se ejecutan un número indeterminado de veces, dependiendo de la condición.

    
    i = 0

    while i < 5:
    print(i)
    i += 1                     #Este bucle va a sumar +1 a i, solo mientras i sea menor a 5.

    #salida: 0 1 2 3 4

    Son utiles porque permiten automatizar tareas repetitivas, ejecutando un bloque de código una y otra vez sin necesidad de repetirlo manualmente. Tambien procesando grandes cantidades de datos como listas, diccionarios o archivos. Y permiten construir algoritmos complejos que se ejecutan de forma eficiente.

    ¿Qué es una lista por comprensión en Python?

    Las listas por comprensión son una forma concisa de crear listas a partir de otras listas, iterables o rangos. Se pueden utilizar para crear listas de números, cadenas, objetos o cualquier otro tipo de dato. La sintaxis básica de una lista por comprensión es la siguiente:

    [ expresión for elemento in iterable if condición ]
  • expresión: La expresión que se evaluará para cada elemento del iterable.
  • elemento: El elemento del iterable que se está iterando.
  • iterable: El iterable que se está iterando.
  • condición: Una condición opcional que se puede utilizar para filtrar los elementos del iterable.

    • Un ejemplo en la creacion de una lista de numeros

    
    numeros_pares = [i for i in range(1, 11) if i % 2 == 0]

    print(numeros_pares)  
    
    #salida [2, 4, 6, 8, 10]

    En el nombre de la variable que almacenará la lista de números pares. En for i in range(1, 11) crea un bucle del 1 al 10 (sin incluir el 11). La variable i toma el valor de cada número en la iteración. Y en if i % 2 == 0 Esta condición verifica si el valor actual de i es divisible por 2. Si es así, se agrega a la lista.

    Un ejemplo en la creacion de una lista de cadenas

    
    cadenas = ["Hola", "mundo", "Python"]

    mayusculas = [cadena.upper() for cadena in cadenas]
    
    print(mayusculas)        

    #salida ['HOLA', 'MUNDO', 'PYTHON']

    En este creamos una nueva cadena con las cadenas en mayuscula, el bucle crea una variable llamada cadena, y busca cada cadena en cadenas y las cambia a mayusculas.

    ¿Qué es un argumento en Python?

    Los argumentos en funciones se refieren a los valores que se pasan a una función para que realice una tarea específica. Como desarrolladores, es común encontrarnos con situaciones en las que necesitamos que una función tome distintos valores para realizar una tarea en particular.

    La flexibilidad y adaptabilidad que ofrecen las funciones Python en el manejo de argumentos es una característica única en el lenguaje, lo que lo convierte en una herramienta poderosa para el desarrollo de aplicaciones complejas. Existen diferentes tipos de argumentos que se pueden pasar a una función en Python:

  • Argumentos posicionales:

    • Este tipo de argumentos se pasan a una función en el orden en que se definieron en la firma de la función.

    
                def suma(a, b):
                    return a + b
                
                resultado = suma(2, 3)
                print(resultado) 
                
                #salida 5

    En este ejemplo, los argumentos a y b son argumentos posicionales y se pasan en el orden en que se definieron en la firma de la función suma.

  • Argumentos con nombre:

    • Los argumentos con nombre se pasan a una función especificando su nombre en la llamada a la función. Este tipo de argumentos proporcionan una mayor claridad y legibilidad al código.

    
                def resta(a, b):
                    return a - b
                
                resultado = resta(b=3, a=5)
                print(resultado) 
                
                #salida 2

    En este ejemplo, los argumentos a y b se pasan con nombre y no en el orden en que se definieron en la firma de la función resta.

  • Argumentos por defecto:

    • Los argumentos con valor por defecto son aquellos que se les asigna un valor en la firma de la función, pero también se pueden sobrescribir en tiempo de ejecución.

    
                def saludo(nombre="Amigo"):
                    return f"Hola, {nombre}!"
                  
                  print(saludo()) 
                
                  # "Hola, Amigo!"

                  print(saludo("Ana")) 
                  
                  # "Hola, Ana!"

    En este ejemplo, el argumento nombre tiene un valor por defecto "Amigo", pero también se puede sobrescribir en tiempo de ejecución.

    ¿Qué es una función Lambda en Python?

    Las funciones lambda o anónimas son un tipo de funciones en Python que típicamente se definen en una línea y cuyo código a ejecutar suele ser pequeño.Tambien se podria decir que, las funciones lambda son simplemente una versión acortada, que puedes usar si te da pereza escribir una función

    Una función que suma dos números como la siguiente.

    
                    def suma(a, b):
                    return a+b

    Pero al usar la funcion lambda queda de la siguiente manera.

    
                    lambda a, b : a + b

    Y es posible llamarla como si fuera una funcion normal

    
                    suma(2, 4)

                    #salida 6

    El poder de lambda se muestra mejor cuando los usa como una función anónima dentro de otra función. Digamos que tiene una definición de función que toma un argumento y ese argumento se multiplicará por un número desconocido.

    
                    def myfunc(n):
                    return lambda a : a * n

    Podemos utilizar esa definición de función para crear una función que siempre duplique el número que envía

    
                    Ejemplo
                    def myfunc(n):
                      return lambda a : a * n
                    
                    mydoubler = myfunc(2)
                    
                    print(mydoubler(11))

                    #salida 22

    En resumen, Las funciones lambda son una herramienta poderosa para escribir código más compacto, legible y simple. Son ideales para tareas pequeñas que no requieren una función completa, mejorando la claridad y la eficiencia del código.

    ¿Qué es un paquete pip?

    El siggnificado de pip es "Package Installer for PythonUn" y es un archivo comprimido que contiene código Python y otros recursos, como archivos de datos, documentación e imágenes. Los paquetes pip se utilizan para distribuir bibliotecas y herramientas de Python de forma organizada y eficiente.

    Características principales:

  • Facilidad de instalación: Los paquetes pip se pueden instalar y actualizar fácilmente usando la herramienta pip, que viene preinstalada en la mayoría de las distribuciones de Python.
  • Reutilización de código: Los paquetes pip permiten compartir código entre diferentes proyectos de Python, lo que facilita el desarrollo y la colaboración.
  • Gestión de dependencias: Los paquetes pip pueden declarar dependencias de otros paquetes, lo que facilita la instalación y gestión de las bibliotecas que necesita tu proyecto.
  • Amplia variedad: Existe una gran cantidad de paquetes pip disponibles en el Python Package Index (PyPI), que es un repositorio central de paquetes de Python.

    • Tipos de paquetes pip:

  • Bibliotecas: Proporcionan código para realizar tareas comunes, como procesamiento de datos, análisis de datos, aprendizaje automático, desarrollo web y mucho más.
  • Herramientas: Proporcionan utilidades para realizar tareas específicas, como formatear código, probar código, crear documentación y mucho más.
  • Marcos de trabajo: Proporcionan una estructura y un conjunto de herramientas para desarrollar aplicaciones web, aplicaciones de escritorio y aplicaciones móviles.

    • Como ejemplo podemos suponer que quieres desarrollar un proyecto de análisis de datos. Puedes instalar la biblioteca NumPy usando pip.

    pip install numpy

    Una vez instalado, se puede importar la biblioteca NumPy del proyecto y comenzar a usarla para realizar operaciones con matrices y vectores.

    Para comprobar si pip está instalado, se abre una ventana de terminal y escribe el siguiente comando:

    pip --version
